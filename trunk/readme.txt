Support for Probabilistic modeling toolkit.
See http://code.google.com/p/pmtksupport/ for details.

In particular, the meta directory contains 'meta' information
documenting what each package does, where it was downloaded from, etc.

If you add a new package, be sure to follow these instructions :
http://code.google.com/p/pmtksupport/wiki/addingNewPackages
